# AI Project Roaster

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Overview

AI Project Roaster is a fun web application where users submit their project descriptions and tech stack, and an AI generates witty roasts based on a selected tone (mild, professional, or apologetic).

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to start roasting projects.

## Features

- Submit project descriptions and tech stack
- Select roast tone: Mild, Professional, or Apologetic
- AI-powered roasting engine
- Responsive design
- Real-time feedback

## Tech Stack

- Next.js 14+
- React 18+
- TypeScript
- Tailwind CSS

## Installation

```bash
npm install
npm run dev
```
